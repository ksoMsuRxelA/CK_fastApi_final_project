# Study Planner API

REST-сервис на FastAPI для планирования учебной траектории студента.

## Стек

- Python 3.12, FastAPI, SQLAlchemy (async)
- PostgreSQL 16
- Docker Compose

## Запуск

```bash
cp .env.example .env
docker compose up --build
```

API: http://localhost:8000, Swagger: http://localhost:8000/docs
